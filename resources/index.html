Congratulations!

Sparta has created an S3-site with a CORS-enabled API Gateway.

CORS Request

The APIGateway has CORS enabled which enables Ajax requests. If a service provisions an API Gateway, Sparta will create two configuration files at the S3 root to support dynamic discovery:

  • sparta_apigateway.js, which can be included as in: 

                  <script src="sparta_apigateway.js"></script>
  • sparta_apigateway.json
